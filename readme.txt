Keyboard Inputs:
ESC to exit the game
A to add a Unit that arrives to the player.
S to add a Unit that seeks to the player.
D to delete a random Unit or if the player is the only remaining unit, to exit the game.

I've included Dean's libraries in my repository. You can find the CPP and H files at \GameAI\steering.
